=== IMAGES UPLOAD DIRECTORY ===

Purpose: Store app logos and icons

Allowed File Types:
- JPEG/JPG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)

File Size Limit: 2MB per image

Naming Convention:
logo_{app_id}_{timestamp}.{extension}
Example: logo_123456_1689345678.png

Directory Permissions: 0755

Notes:
- Images are automatically resized if needed
- Square aspect ratio (1:1) recommended
- Transparent background PNGs preferred for logos

For Admin: This directory is auto-created during setup.
