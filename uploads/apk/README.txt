=== APK UPLOAD DIRECTORY ===

Purpose: Store Android APK files

Allowed File Types:
- APK (.apk) only

File Size Limit: 200MB per APK

Naming Convention:
app_{app_id}_{timestamp}.apk
Example: app_123456_1689345678.apk

Security Notes:
- All APKs are scanned for viruses
- Only verified MODs are accepted
- Keep backups of original APKs

Directory Permissions: 0755

For Admin: This directory is auto-created during setup.
